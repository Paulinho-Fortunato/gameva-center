
# GAMEVA CENTER

> Transformamos o seu PC em Console PlayStation em Angola

## 📋 Sobre

**GAMEVA CENTER** é uma empresa especializada em tecnologia gamer que transforma computadores em consoles PlayStation (PS3, PS4) funcionais. Oferecemos soluções tecnológicas inovadoras e seguras, com atendimento profissional e suporte dedicado em Angola.

---

## 🎮 Serviços

- **Transformação PC em Console** — Converta seu computador em PS3 ou PS4
- **Instalação de Jogos** — Biblioteca de jogos configurada e otimizada
- **Venda de CPU** — Computadores acessíveis prontos para transformação
- **Venda de Comandos** — Controladores PlayStation compatíveis
- **Suporte Técnico** — Atendimento domiciliar e online via WhatsApp

---

## 🛠 Stack Tecnológico

- **Backend:** Hono.js (TypeScript)
- **Frontend:** HTML5, CSS3, JavaScript Vanilla
- **Animações:** AOS (Animate On Scroll), Custom CSS Animations
- **Fonts:** Orbitron, Rajdhani, Inter (Google Fonts)
- **Icons:** FontAwesome 6.5.0
- **Deployment:** Static serving via Hono

## 🚀 Instruções de Uso

### Instalação

```bash
npm install
```

### Desenvolvimento

```bash
npm run dev
```

Acede a `http://localhost:3000`

### Produção

```bash
npm run build
npm start
```
